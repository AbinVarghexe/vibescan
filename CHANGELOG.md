## [0.1.0] - 2026-03-02

### Added
- 🎉 Initial release of VibeScan
- **Dependency Scanning:** Automatic detection of `package.json` (NPM) and `requirements.txt` (PyPI)
- **Hallucination Detection:** Flags non-existent packages generated by AI coding assistants
- **Typosquatting Protection:** Identifies malicious packages impersonating legitimate libraries
- **Risk Scoring System:** 0-100 risk score based on package age, downloads, and registry availability
- **Actionable Reports:** Categorizes dependencies into Safe, Suspicious, and Critical risks
- **CI/CD Integration:** Exit codes for automated build pipelines
- **CLI Interface:** Easy-to-use command-line tool (`vibescan`)
- **Web Interface:** Streamlit-based web application for interactive scanning
- **Docker Support:** Containerized deployment option
- **NPM Wrapper:** JavaScript/Node.js integration via `vibescan-js`
- **Comprehensive Documentation:** Installation guides, usage examples, and API reference

### Features
- Multi-registry support (PyPI and NPM)
- Configurable risk thresholds
- Detailed risk reports with actionable recommendations
- Color-coded terminal output for better visibility
- JSON output format for tool integration
- Batch scanning capabilities
- Offline package validation
- Similarity detection for typosquatting
- Registry availability checks
- Package metadata analysis

### Security
- No external dependencies on third-party AI services
- Local-first scanning approach
- Secure API calls to official package registries
- No data collection or telemetry
