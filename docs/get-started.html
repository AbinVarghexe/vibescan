Introduction

VibeScan is your first line of defense against AI hallucinations and supply chain attacks.

As developers increasingly rely on LLM code generators (ChatGPT, Copilot, Claude), the risk of introducing non-existent ("hallucinated") or malicious packages has skyrocketed. VibeScan automatically detects these issues before you install them.

Installation

VibeScan is available as a Python package. It's recommended to install it globally or in your virtual environment.

From PyPI

# Install using pip
pip install vibescan

From Source

# Clone the repository
git clone https://github.com/AbinVarghexe/vibescan.git
cd vibescan

# Install in editable mode
pip install -e .

Basic Usage

Navigate to your project directory (where your package.json or requirements.txt is located) and run: 

# Scan the current directory
vibescan

To scan a specific directory path: 

vibescan /path/to/your/project

To enable verbose debug logging: 

vibescan --debug

CI/CD Integration

Automate your security checks by adding VibeScan to your GitHub Actions pipeline. 

name: Security Scan
on: [push, pull_request]

jobs:
  vibescan:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-python@v4
        with:
          python-version: '3.x'
      - name: Install VibeScan
        run: pip install vibescan
      - name: Run Scan
        run: vibescan
