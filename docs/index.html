VibeScan

A powerful pre-publish security analysis tool that detects AI hallucinated dependencies and slopsquatting before they enter your codebase.

Try It Now View on GitHub
$ pip install vibescan
$ npm install -g vibescan-js
Visit vibescen.streamlit.app
🔍

Hallucination Detection

Verifies every dependency exists in npm and PyPI registries, catching AI-generated phantom packages before they become attack vectors.

🎭

Typosquatting Defense

Advanced similarity matching detects names like reqeusts mimicking real packages to slip malware into builds.

📊

Explainable Risk Scores

Every dependency gets a transparent 0–100 risk score with clear reasoning — no black-box verdicts.

⚡

Fast & Local

Scans finish in seconds without sending source code anywhere. Your codebase stays private.

🔄

Cross-Ecosystem

Supports package.json and requirements.txt — covering npm and PyPI in one tool.

🛡️

CI/CD Ready

Drop into GitHub Actions, GitLab CI, or pre-commit hooks. Automate security scanning across your entire pipeline.
