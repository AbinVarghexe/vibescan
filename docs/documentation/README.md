# VibeScan Documentation

Welcome to the VibeScan documentation folder. This directory contains comprehensive guides and technical documentation for understanding and using VibeScan.

## 📚 Documentation Files

### [USAGE.md](USAGE.md)
Complete guide on how to use VibeScan including:
- Installation methods (pip, npm, Docker, source)
- Command-line usage and flags
- Integration with CI/CD pipelines (GitHub Actions, GitLab CI, Pre-commit hooks)
- Configuration options and risk scoring thresholds
- Output interpretation and examples

### [CODE_EXPLANATION.md](CODE_EXPLANATION.md)
Technical deep-dive into VibeScan's architecture:
- Project structure and module organization

- Registry API integration details
- Risk scoring algorithm breakdown
- Extension and customization guidelines
